## New Fields Information

### lecturer_name
```
Type:       CharField (max 100 characters)
Required:   No (can be left blank)
Example:    "Dr. Jane Smith", "Prof. John Doe"
Display:    Dashboard, Session Detail
Input:      Text field in Create Session form
Purpose:    Identify lecturer for each session
```

### class_year
```
Type:       CharField with choices
Required:   Yes (must select)
Options:    Year 1, Year 2, Year 3, Year 4, Year 5
Display:    Dashboard, Session Detail
Input:      Dropdown selector in Create Session form
Purpose:    Specify which class level is being taught
```
